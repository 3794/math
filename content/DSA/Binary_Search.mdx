## Binary Search

정렬된 array에서 특정 아이템을 찾을 때 사용됩니다. 배열의 중간 인덱스를 찾아 타겟과 비교하여 타겟이 중간 인덱스보다 작으면 왼쪽 배열을, 타겟이 중간 인덱스보다 크면 오른쪽 배열을 탐색합니다. 이 과정을 반복하여 타겟을 찾습니다. Linear search보다 더 빠른 $O(log n)$의 시간복잡도를 가집니다.

```typescript
function binarySearch(arr: number[], target: number): number {
  let low = 0;
  let mid;
  let high = arr.length - 1;

  while (low <= high) {
    mid = Math.floor((low + high) / 2);
    if (arr[mid] === target) return mid;
    if (arr[mid] < target) low = mid + 1;
    else high = mid - 1;
  }
  return -1;
}
```

Recursive 방식으로 구현할 수도 있습니다.

```typescript
function binarySearch(arr: number[], target: number): number {
  function binarySearchHelper(
    arr: number[],
    target: number,
    low: number,
    high: number
  ): number {
    if (low > high) return -1;
    const mid = Math.floor((low + high) / 2);
    if (arr[mid] === target) return mid;
    if (arr[mid] < target)
      return binarySearchHelper(arr, target, mid + 1, high);
    else return binarySearchHelper(arr, target, low, mid - 1);
  }
  return binarySearchHelper(arr, target, 0, arr.length - 1);
}
```
