# Heap (priority queue)

- https://leetcode.com/problem-list/heap-priority-queue/
- [2974. Minimum Number Game
](https://leetcode.com/problems/minimum-number-game/)

**Priority Queue(우선순위 큐)** 는 추상 자료형이고, **Heap(힙)** 은 그 Priority Queue를 효율적으로 구현하는 대표적인 방법입니다

| 구분 | Heap | Priority Queue (우선순위 큐) |
| :-- | :-- | :-- |
| 개념 | 완전 이진 트리 기반의 자료구조 | "우선순위"가 높은(작은/큰) 값부터 꺼냄 |
| 역할 | 최솟값(또는 최댓값) 빠르게 추출 | 우선순위에 따라 삽입/삭제/조회 |
| 구현 | 배열, 트리 등으로 직접 구현 | Min-Heap, Max-Heap 등 여러 구조로 구현 |
| 시간복잡도 | 삽입/삭제 O(log n) | (힙 기반일 때) 삽입/삭제 O(log n) |

파이썬에서는 ```heapq.heapify(nums)``` 를 사용한다.

예시:
```python
import heapq

heapq.heapify(nums)      # 배열을 최소 힙으로 변환

min = heapq.heappop(nums) 
```

js에서는 직접 구현하거나 외부 라이브러리를 사용해야 합니다.

예시:
```typescript
class MinHeap {
    private heap = []
    
    size() {
        return this.heap.length;
    }

    peek() {
        return this.heap[0];
    }

    push(val) {
        this.heap.push(val);
        this._bubbleUp();
    }

    pop() {
        if (this.size() === 1) return this.heap.pop();
        const min = this.heap[0];
        this.heap[0] = this.heap.pop();
        this._bubbleDown();
        return min;
    }

    _bubbleUp() {
        let idx = this.heap.length - 1;
        while (idx > 0) {
            let parent = Math.floor((idx - 1) / 2);
            if (this.heap[parent] <= this.heap[idx]) break;
            [this.heap[parent], this.heap[idx]] = [this.heap[idx], this.heap[parent]];
            idx = parent;
        }
    }

    _bubbleDown() {
        let idx = 0;
        const length = this.heap.length;
        while (true) {
            let left = 2 * idx + 1;
            let right = 2 * idx + 2;
            let smallest = idx;

            if (left < length && this.heap[left] < this.heap[smallest]) smallest = left;
            if (right < length && this.heap[right] < this.heap[smallest]) smallest = right;
            if (smallest === idx) break;
            [this.heap[smallest], this.heap[idx]] = [this.heap[idx], this.heap[smallest]];
            idx = smallest;
        }
    }
}
```
