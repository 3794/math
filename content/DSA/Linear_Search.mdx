## Linear Search

정렬되지 않은 array에서 특정 아이템을 찾을 때 사용됩니다. array element들을 하나 하나씩 비교하여 찾습니다. $linear$ 혹은 $O(n)$의 시간복잡도를 가집니다.

```typescript
function linearSearch(arr: number[], target: number): number {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) return i;
  }
  return -1;
}
```
