## Two Pointers

Array를 순회할때 보통 **하나의 포인터**를 가지고 첫번째 element부터 마지막 element까지 순회하는 방법을 사용합니다. 하지만 때때로 **두개의 포인터**를 동시에 사용하는것이 필요한 때가 있습니다.

elements를 반대로 뒤집는 예제를 살펴봅시다.

javascript에서는 배열의 reverse 메서드를 사용할 수 있습니다.

```typescript
const arr = [1, 2, 3, 4, 5];
arr.reverse();
```

java에서는 array를 List로 변환해야 List의 reverse 메서드를 사용할 수 있습니다.

```java
public static void reverseArray(Integer[] arr) {
    List<Integer> list = Arrays.asList(arr);
    Collections.reverse(list);
}
```

List로 변환하는 방법을 사용하지 않기 위해서는 새로운 배열을 생성하고, 원본 배열을 뒤에서 읽어 새로운 배열에 순서대로 넣어야 합니다.

```java
public static int[] reverseArray(int[] arr) {
    int[] result = new int[arr.length];
    for (int i = 0; i < arr.length; i++) {
        result[i] = arr[arr.length - 1 - i];
    }
    return result;
}
```

Two Pointers를 사용하면 $O(1)$의 공간복잡도로 문제를 해결할 수 있습니다.

```java
public static void reverseArray(int[] arr) {
    int left = 0;
    int right = arr.length - 1;

    while (left < right) {
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        left++;
        right--;
    }
}
```

typescript:

```typescript
function reverseArray(arr: number[]): number[] {
  let left = 0;
  let right = arr.length - 1;

  while (left < right) {
    const temp = arr[left];
    arr[left] = arr[right];
    arr[right] = temp;
    left++;
    right--;
  }

  return arr;
}
```
