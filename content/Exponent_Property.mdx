# Exponent Property

Exponent Property는 지수의 곱셈과 나눗셈을 계산하는 데 사용됩니다.

## 예제

| Property                                    | Example                                      |
| ------------------------------------------- | -------------------------------------------- |
| $x^n\cdot x^m=x^{n+m}$                      | $2^3\cdot 2^5=2^8$                           |
| $\dfrac{x^n}{x^m}=x^{n-m}$                  | $\dfrac{3^8}{3^2}=3^6$                       |
| $\left(x^n\right)^m=x^{n\cdot m}$           | $\left(5^4\right)^3=5^{12}$                  |
| $(x\cdot y)^n=x^n\cdot y^n$                 | $(3\cdot 5)^7=3^7\cdot 5^7$                  |
| $\left(\dfrac xy\right)^n=\dfrac{x^n}{y^n}$ | $\left(\dfrac{2}3\right)^5=\dfrac{2^5}{3^5}$ |

## Negative Exponent

$$
x^{-n}=\dfrac{1}{x^n}
$$

$$
\dfrac{1}{x^{-n}}=x^n
$$
