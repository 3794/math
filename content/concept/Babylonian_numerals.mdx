# 바빌로니아 숫자 (Babylonian Numerals)

바빌로니아는 BC 2000년경 **60진법(Sexagesimal System)** 을 사용한 진법 체계를 개발했다. 이는 현대의 분, 초, 각도 측정에서도 사용된다.

## 특징

### 60진법

- 기수: 60
- 한 자리는 0부터 59까지의 값을 나타냄
- 오늘날 시간(60초), 각도(60분) 측정에 남아있음

### 기호

- **1-59**: 설형문자(쐐기 모양)로 표기
  - 수직 쐐기(▼): 1
  - 꺾인 쐐기(◄): 10
  - 예: 23 = ▼▼◄▼▼▼

### 위치 기수법 (Positional Notation)

- 오른쪽에서 왼쪽으로 60의 거듭제곱 적용
- 예: 2, 30, 45 = 2×60² + 30×60 + 45 = 9045

**문제**: 0의 개념이 불명확했고, 맥락에 따라 해석이 달랐음

## 의의

- ✓ 최초의 본격적인 진법 체계
- ✓ 위치 기수법의 선구
- ✓ 천문학, 수학에서 정밀한 계산 가능
- ✓ 현대에도 시간과 각도 측정에 사용
- ✗ 큰 수 표기에 불편
- ✗ 0이 없어 표기 모호함

## 예시

| 현대 표기 | 60진법 표기 | 계산             |
| --------- | ----------- | ---------------- |
| 1         | 1           | 1                |
| 61        | 1, 1        | 1×60 + 1         |
| 3661      | 1, 1, 1     | 1×60² + 1×60 + 1 |
| 125       | 2, 5        | 2×60 + 5         |
