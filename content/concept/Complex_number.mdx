# Complex Number (복소수)

## 복소수란?

**복소수**는 실수와 허수를 합친 수이다.

$$z = a + bi$$

여기서:

- $a$: 실수부 (Real part)
- $b$: 허수부 (Imaginary part)
- $i$: 허수 단위 ($i^2 = -1$)

**예시:**

- $3 + 2i$ → 실수부: 3, 허수부: 2
- $5 - 4i$ → 실수부: 5, 허수부: -4
- $2i$ → 순허수 (실수부가 0)
- $7$ → 실수 (허수부가 0)

## 허수 단위 i

$$i^2 = -1$$
$$i^3 = -i$$
$$i^4 = 1$$

이 규칙이 반복되므로, $i^n$의 값은 $n$을 4로 나눈 나머지에 따라 결정된다.

## 복소수의 계산

### 덧셈과 뺄셈

$$(a + bi) + (c + di) = (a+c) + (b+d)i$$

예: $(3 + 2i) + (1 + 4i) = 4 + 6i$

### 곱셈

$$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$$

**주의:** $i^2 = -1$을 사용한다.

예: $(2 + 3i)(1 + 2i) = 2 + 4i + 3i + 6i^2 = (2-6) + (4+3)i = -4 + 7i$

### 켤레복소수

$a + bi$의 켤레복소수는 $a - bi$이다. 기호: $\overline{a + bi} = a - bi$

**용도:** 나눗셈할 때 사용
$$\frac{a + bi}{c + di} = \frac{(a+bi)(c-di)}{(c+di)(c-di)}$$

## 복소평면

복소수를 평면에 나타낼 수 있다:

- 가로축: 실수부
- 세로축: 허수부

예를 들어, $3 + 2i$는 $(3, 2)$ 위치에 점으로 표시된다.

## 이차방정식에서의 활용

판별식이 음수인 경우, 복소수 해가 나타난다.

**예:**
$$x^2 + 1 = 0$$
$$x^2 = -1$$
$$x = \pm i$$
