# Discrete Mathematics(이산수학)

## 정의

이산수학은 **연속적이지 않은, 띄엄띄엄한 값들을 다루는 수학**이다.

"이산"이란 쉽게 말해 **나뉘어진, 분리된**이라는 뜻이다.

## 연속수학 vs 이산수학

| 구분         | 설명                  | 예시                         |
| ------------ | --------------------- | ---------------------------- |
| **연속수학** | 부드럽게 이어진 값들  | 1.5, 2.7, 3.14159... (실수)  |
| **이산수학** | 떨어져 있는 개별 값들 | 1, 2, 3, 4... (자연수, 정수) |

## 실생활 예시

### 연속수학이 필요한 경우

- 물리학: 물체의 움직임, 속도, 가속도 (시간은 계속 흐름)
- 미적분: 그래프가 부드럽게 이어짐

### 이산수학이 필요한 경우

- 컴퓨터: 0과 1만 존재
- 암호화: 정수만 사용
- 조합론: "사람 3명 중 2명을 선택하는 경우의 수는?"
- 그래프 이론: SNS의 친구 관계망

## 이산수학의 주요 영역

### 1. 논리학 (Logic)

참과 거짓만 있는 명제를 다룬다.

- 예: "비가 오면 우산을 쓴다" (참 또는 거짓)

### 2. 집합론 (Set Theory)

서로 다른 원소들의 모임을 다룬다.

- 예: `{1, 2, 3}`, `{빨강, 파랑, 노랑}`

### 3. 조합론 (Combinatorics)

선택하고 배열하는 경우의 수를 다룬다.

- 예: 10명 중 3명을 선택하는 경우의 수는?

### 4. 그래프 이론 (Graph Theory)

점(노드)과 선(간선)으로 이루어진 구조를 다룬다.

- 예: 지하철 노선도, 도시 간 거리

### 5. 수열과 급수 (Sequences and Series)

순서가 있는 개별 숫자들을 다룬다.

- 예: 1, 2, 4, 8, 16... (등비수열)

### 6. 정수론 (Number Theory)

정수들의 성질과 관계를 다룬다.

- 예: 소수, 최대공약수, 최소공배수
