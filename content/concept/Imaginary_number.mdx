## Imaginary number, 허수

#### imaginary number 개념의 역사

- 고대 그리스의 [헤론](/people/Hero)이 imaginary number 개념의 초기 형태를 언급한 기록이 있다.
- **16세기 이탈리아의 수학자 지롤라모 카르다노(Geronimo Cardano)** 가 삼차방정식의 해법을 연구하는 과정에서 음수의 제곱근, 즉 imaginary number의 필요성을 인식하고 실제 계산에 사용했다.
  그의 저서 『아르스 마그나(Ars Magna, 1545)』는 imaginary number가 등장하는 최초의 책으로 꼽힌다.
- 카르다노는 imaginary number를 “가상적인 수”로 여기며, 수로서 완전히 받아들이지는 않았지만, 방정식의 해를 구하는 과정에서 imaginary number의 개념을 도입했다.
- 17세기 [데카르트](/people/Descartes) 가 “imaginary number”라는 용어를 처음 사용했다.
- 이후 오일러, 가우스, 월리스, 베셀 등 여러 수학자들이 imaginary number와 복소수의 이론적 기초를 다져, 오늘날의 체계가 완성되었다.

#### 상세 설명

    -	**실수(Real Number)** 의 세계에서는 어떤 수를 제곱해도 0 이상(음수가 아님)이 된다.
    -	예: $2^2 = (-2)^2 = 4$,
    -	음수의 제곱근(예: $\sqrt{-1}$)은 실수 범위에서는 존재하지 않는다.
    -	수학자들은 이 문제를 해결하기 위해 **imaginary number** 를 정의했다.

#### imaginary number 정의

- $i$는 다음과 같이 정의된다:
  $i^2 = -1$
- 즉, $i$는 제곱해서 -1이 되는 수이다.
- $\sqrt{-4} = 2i$ 처럼, 제곱해서 음수가 되는 모든 수는 imaginary number로 표현할 수 있다.
